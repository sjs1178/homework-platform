# Product Brief

## App Name

homework-platform

## One-line Description

부모와 자녀가 함께 숙제 일정을 캘린더로 관리하고, 완료 시 리워드를 받는 패밀리 플랫폼.

## Target User

- **부모**: 자녀 숙제를 직접 입력·관리하고 싶은 학부모
- **자녀**: 오늘 해야 할 숙제를 한눈에 보고 완료 처리하고 싶은 초등~중등
- 사용자가 겪는 문제: 숙제 관리가 산발적이고, 자녀가 스스로 확인·완료하는 습관이 없음
- 현재 대안: 카카오톡, 알림장 앱, 수기 메모
- 우리 앱을 써야 하는 이유: AI로 자연어/이미지 입력 → 자동 일정화 + 리워드로 자녀 자기주도 학습 유도

## Core Value Proposition

이 앱은 **학부모**가 **AI를 통해 숙제를 빠르게 일정화**하고, **자녀가 스스로 완료 처리하며 리워드를 적립**할 수 있게 해준다.

## MVP Scope

### Must Have

- Google 계정 로그인
- 부모-자녀 페어링 (초대 코드 방식)
- 부모: 자연어/이미지로 숙제 입력 → Claude API 파싱 → 캘린더 등록
- 자녀: 월간/일간 캘린더로 숙제 조회
- 자녀: 숙제별 완료 버튼 → 리워드 자동 적립
- 부모: 리워드 종류 설정 (게임시간/용돈포인트 + 명칭 + 단위)
- 자녀: 누적 리워드 및 사용 내역 조회

### Should Have

- 숙제 완료 시 부모에게 알림 (이메일 또는 푸시)
- 부모: 리워드 수동 추가/차감
- 과목별 반복 입력 규칙 설정 (homework-alert의 rules 시스템 참고)
- 다크모드

### Not Now

- 모바일 앱 (React Native / Expo) — 웹 MVP 검증 후
- 다중 자녀 지원 (MVP는 1쌍)
- 학원 연동
- 소셜 공유

## Success Metrics

- Activation: 가입 후 숙제 3개 이상 등록
- Retention (D7): 7일 후 재방문율
- Engagement: 자녀 완료 처리율 (등록 대비 완료 %)
- Revenue: TBD (MVP는 무료)
