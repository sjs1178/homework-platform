# Requirements

## 기능 요구사항

### 인증 / 페어링

- [ ] Google OAuth 로그인 (Supabase Auth)
- [ ] 가입 시 역할 선택: 부모 / 자녀
- [ ] 부모: 초대 코드 생성 및 공유
- [ ] 자녀: 초대 코드 입력으로 부모와 페어링

### 부모 — 숙제 입력

- [ ] 자연어 텍스트 입력 → Claude API → HomeworkItem 파싱
- [ ] 이미지 첨부 → Claude Vision API → HomeworkItem 파싱
- [ ] 파싱 결과 미리보기 후 확인/수정
- [ ] 과목별 입력 규칙 설정 (subject rules)
- [ ] 등록된 숙제 수정·삭제

### 부모 — 리워드 설정

- [ ] 리워드 종류 1: 게임시간 (분 단위, 명칭 설정 가능)
- [ ] 리워드 종류 2: 용돈 포인트 (단위·명칭 설정 가능)
- [ ] 숙제별 완료 리워드 금액 설정
- [ ] 리워드 수동 추가/차감 (부모 권한)

### 자녀 — 캘린더 조회

- [ ] 월간 캘린더 뷰: 날짜별 숙제 개수 표시
- [ ] 일간 뷰: 시간대별 숙제 리스트
- [ ] 숙제별 완료 버튼
- [ ] 완료 처리 → 리워드 즉시 적립

### 자녀 — 리워드 조회

- [ ] 총 적립 리워드 표시
- [ ] 사용 내역 조회
- [ ] 잔여 리워드 표시

## 비기능 요구사항

- 성능: 페이지 로드 < 2초 (Vercel Edge)
- 보안: RLS(Row Level Security)로 부모-자녀 데이터 격리
- 반응형: 모바일 웹 우선 (향후 앱 전환 고려)
- AI: Anthropic Claude API (claude-sonnet-4-6 기본, prompt caching 적용)
