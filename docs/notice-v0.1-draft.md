## 주요 기능 안내

### 📋 AI 숙제 등록
부모님이 자연어로 말하듯 입력하거나, 숙제 사진을 찍으면 AI가 자동으로 과목·내용·마감일을 파악해 일정으로 등록합니다.

### ✅ 자녀 숙제 완료 & 검사
자녀가 숙제를 완료하고 풀이 사진을 올리면, AI가 자동으로 채점합니다. 부모님이 결과를 검토하고 확정하면 학습 데이터로 쌓입니다.

### 🎁 리워드 시스템
숙제를 완료하거나 좋은 점수를 받으면 리워드가 자동으로 적립됩니다. 부모님이 리워드 이름과 단위를 자유롭게 설정할 수 있어요.

### 🔗 패밀리 연결
초대 코드 하나로 부모와 자녀를 연결합니다. 여러 자녀, 여러 보호자도 함께 관리할 수 있습니다.

### 📊 학습 통계
과목별 완료 현황과 성취도를 한눈에 확인할 수 있습니다.

## AI 기능 이용 방법

kiddoloop의 AI 기능은 두 가지 방법으로 이용할 수 있습니다.

1. **본인 API 키 사용 (무제한)**: 설정 → AI 설정에서 Claude, ChatGPT, Gemini API 키를 입력하면 AI 기능을 무제한으로 이용할 수 있습니다. API 키는 이 기기에만 저장되며 서버에 기록되지 않습니다.

2. **광고 시청 후 이용**: API 키 없이도 광고를 시청한 후 AI 기능을 1회 이용할 수 있습니다.
