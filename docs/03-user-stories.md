# User Stories

## 부모

- As a parent, I want to sign in with Google, so that I don't need a separate account.
- As a parent, I want to generate an invite code, so that I can pair with my child.
- As a parent, I want to type a natural language message like "수학 학습지 내일까지", so that Claude automatically creates a homework item.
- As a parent, I want to upload a photo of a homework sheet, so that Claude extracts the assignments automatically.
- As a parent, I want to preview and edit the parsed homework before saving, so that I can correct mistakes.
- As a parent, I want to set a reward type (game time or points) with a custom name, so that the reward is meaningful to my child.
- As a parent, I want to see when my child completes homework, so that I can track progress.
- As a parent, I want to set per-subject parsing rules, so that recurring assignments are handled consistently.

## 자녀

- As a child, I want to sign in with Google and enter an invite code, so that I can connect to my parent.
- As a child, I want to see my homework on a monthly calendar, so that I know what's coming.
- As a child, I want to see today's homework in a time-ordered list, so that I know what to do now.
- As a child, I want to tap a "완료" button on each homework item, so that I can mark it done and earn a reward.
- As a child, I want to see my total accumulated reward balance, so that I know how much I've earned.
- As a child, I want to see my reward usage history, so that I can track what I've spent.
